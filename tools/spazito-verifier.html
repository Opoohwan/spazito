Spazito Verifier

Offline check that a price text is genuinely from your bot · no network · self-test…

How this works & how to trust it 
This page makes NO network requests - verify by reading the source, or run it in
airplane mode. Your key never leaves this device.

Each text ends with  [#N TAG]  where TAG is the first 8 hex characters (uppercase)
of  HMAC-SHA256( key , "N|PAYLOAD" ) ,  and PAYLOAD is the message text with the
[#N TAG] block removed and trimmed. The bot and this tool compute the exact same
thing; if the codes match, the text is authentic and unaltered. If any character
of the prices changed, or the code was faked without the key, it fails.

The # count must climb by 1 each weekday. A repeat = replay; a jump = missed texts.

Self-test: on load this page computes a known HMAC-SHA256 test vector and shows
PASS only if the browser's crypto returns the correct answer.
